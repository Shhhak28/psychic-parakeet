### Описание
Программа на Python для генерации случайных паролей. Можно настраивать длину и состав символов. Все пароли сохраняются в историю.

### Функционал
- Настройка длины через ползунок (4-32).
- Выбор состава: цифры, буквы, знаки.
- Автоматическое сохранение в `password_history.json`.
- Таблица с историей последних генераций.

### Как запустить
1. Установите Python.
2. Запустите файл командой: `python main.py`.

### Тесты
- Если не выбрать ни один чекбокс, программа выдаст предупреждение.
- При каждом нажатии кнопки "Сгенерировать" новая запись появляется в таблице.
