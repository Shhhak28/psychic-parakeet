### Описание
Приложение на Python (Tkinter) для быстрого поиска информации о пользователях GitHub и сохранения их в локальный список избранного.

### Как использовать API
Программа использует публичное **GitHub REST API**
- Запросы выполняются методом GET.
- Для базового использования (до 60 запросов в час) авторизация не требуется.

### Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install requests`.
3. Запустите файл: `python main.py`.

### Примеры тестов
- **Пустой ввод:** Оставьте поле пустым и нажмите "Найти" — должно появиться предупреждение.
- **Несуществующий юзер:** Введите случайный набор символов — программа сообщит, что пользователь не найден.
- **Избранное:** Найдите `torvalds`, нажмите "В избранное", перезапустите программу и нажмите "Показать избранное" — данные должны сохраниться.
